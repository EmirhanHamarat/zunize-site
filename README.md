**Zunize**

Zunize is a premium handmade chocolate brand offering carefully crafted collections with a modern and elegant shopping experience.

*Features

- Curated chocolate box collections
- Product-focused shopping experience
- Fast and responsive UI (Next.js)
- Minimal and premium design

*Tech Stack

- Next.js 14
- React
- TypeScript
- Tailwind CSS

*Installation

Live - https://zunize.com
Developed by Emirhan Hamarat

```bash
npm install
npm run dev
